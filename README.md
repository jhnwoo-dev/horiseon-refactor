# horiseon-refactor

## Description

This project is a refactoring of an existing website code. The main purpose of this project was to practice the removal of redundant, non-semantic items of HTML code on this existing website and replace them with semantic elements of HTML that makes logical structural sense. This allows future developers who may look at this website to have a better understanding of what the certain portions of the website were meaning to be.
